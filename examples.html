SQL Query

This is a SQL query which will return 3 records which include the name of the director and title of the movie with the best ratings.

SELECT movie.mov_title AS "Title", director.dir_fname || ' ' || director.dir_lname AS "Director's Name", rating.rev_stars AS "rating" FROM movie INNER JOIN movie_direction ON movie.mov_id = movie_direction.mov_id INNER JOIN director ON movie_direction.dir_id = director.dir_id INNER JOIN (select * FROM rating WHERE rating IS NOT NULL) rating ON movie.mov_id = rating.mov_id ORDER BY rating DESC LIMIT 3;

The query begins by selecting the movie title from the movie table and then concatenating the first and last names of the director, as these are in separate columns in the directors table. The names of the columns are also renamed using the AS keyword.

The query then joins the movie_direction table, which links the director and movies tables using their respective id's.

The rating table is then joined to the to add the movie reviews by movie id using a subquery which filters out any reviews which have no rating.

Finally the three movies with the best ratings are returned, in descending order (i.e. highest first).
